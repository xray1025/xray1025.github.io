度小满笔试2.昆虫走迷宫

发表于 2020-09-21 分类于 笔试

  • @代表昆虫位置，.代表可以通行，#代表不可越过的墙壁，*代表可以越过的墙壁，求昆虫走出迷宫最小破壁次数

    • 输入：

      3

      3 3

      ###

      *@#

      .##

      3 3

      @##

      …

      #**

      3 3

      #.#

      *@#

      ###

    • 输出：

      1

      0

      1
