度小满笔试2.昆虫走迷宫

Author: X_ray Date: 九月 21, 2020  9:09:23 Category: 笔试

  • @代表昆虫位置，.代表可以通行，#代表不可越过的墙壁，*代表可以越过的墙壁，求昆虫走出迷宫最小破壁次数

    • 输入：

      3

      3 3

      ###

      *@#

      .##

      3 3

      @##

      …

      #**

      3 3

      #.#

      *@#

      ###

    • 输出：

      1

      0

      1
